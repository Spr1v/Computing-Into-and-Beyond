Data, Information, and Information Systems

"Data transforms into information through intelligent systems."
Data

Raw information that can be stored electronically, it can come in many forms such as words, numbers, etc that is interpreted by a machine, the computer
Information

Processed information that helps both the user and the computer to initialize decision making crucial for the function and the output.
Different Types of Support Systems in Information Systems
Uses of Information System
Decision Making Aids the decisions, by assessing the given data by the user.

Data Collection Collects all raw data that will be processed later on.

Sorting and Storing data The processed data will be used by assigned categories, or stored for later use.

Operations Management Manages the data such as expenses, inventory, procuring goods.

Information Access Easy access to a company/business data that can be used any way.
