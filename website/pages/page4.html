Computer Hardware and Software: Their Implementation in the different institutions in the community

"Hardware and software transform communities through innovation."
Internal Hardware

Installed inside the computers case, this is what powers the computer system to run and other primary functions
Motherboard

- Processing system that connects the important hardwares together

- CPU slots, Memory slots, Integrated functions needed for the computer system
CPU

- Central Processing Unit

- Brain of the computer that manages every action and data that flow in and out of the computer
RAM (Random Access Memory)

- Storage of instructions before execution

- Uses the computers power to maintain function
Graphics Card

- Renders the animations to be displayed

- GPU quickly accesses videos through video memory
External Hardware

Connected to the computer, this allows the interaction of Computer and the User
Monitor

- Comes in different resolutions, sizes, but ultimately displays processed information

- Displays videos, texts, photos
Speaker

- Plays the audio associated with the display on the screen

- Sounds help visualize the displayed content
Mouse

- Lets users click and interact with the displayed items in the monitor

- Has buttons for moving, deleting, selecting, scrolling
Keyboard

- Lets users input letters and words into the computer through the interactable letters

- GPU quickly accesses videos through video memory
Application Software

Programs that function based on the user tasks
General Purpose Application Software

-Programs that provide common information to the user such as browing, presenting, which is flexible and has many purposes
Application specific software

- Programs that a completes a specific tasks especially for business applications. These are extremely efficient, but are not suited for other tasks.
System Software

Enables the computer to start and function, ensuring it properly operates
System Management Software

- Manages, runs, supports the hardware and executes the program in order to start the computer
System Development Software

- Defines, and tests the software developed to ensure that the program is maintained /up to date and supported by all devices
